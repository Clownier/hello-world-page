# Hello World Page 🌍

一个简单有趣的个人主页，展示 Hello World 并包含互动小游戏。

## 功能

- 🌍 Hello World 标题 + 彩虹渐变动画
- 📅 实时日期时间显示（中文格式）
- 🖱️ 鼠标彩虹拖尾特效
- 📍 IP 地址定位显示
- 🎮 三款互动小游戏：
  - 🎆 烟花：自动释放 + 点击释放
  - 🫧 泡泡：30秒点击挑战
  - ⚡ 反应：测试反应速度

## 项目结构

```
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── games.js        # 游戏逻辑
├── VERSION.md          # 版本历史
└── README.md           # 项目说明
```
